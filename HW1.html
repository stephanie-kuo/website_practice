3 Tips to Write HTML Super Fast

Basic Settings of Sublime3 (Portable) for HTML

Must Have: Sublime3(portable), node.js, Internet connection

Tip1: HTML-CSS-JS Prettify

It can do this

    Prettify your code by ctrl+s

STEP1: Install Package Control

1. Search "Package Control" on Google

2. Copy the codes

3. Open your Sublime3 and press ctrl+~
Then paste the codes and press enter

STEP2: Install "HTML-CSS-JS Prettify"

1. Ppress ctrl+shift+p, key in "pci", select "Package Control: Install Package"

2. Search "htmlcssjs" and select "HTML-CSS-JS Prettify"

STEP3: Test

1. Create a new HTML file and write codes.

2. Press ctrl+s. Successful!!
